# รูปสำหรับ starter pack "ของเล่น / แม่และเด็ก"

วางไฟล์ .jpg ตามรายการนี้ให้ครบ แล้ว deploy — ตัวเลือก "ของเล่น / แม่และเด็ก" ในหน้า signup
จะเปิดให้เลือกอัตโนมัติ (ระบบเช็คไฟล์ใน `lib/starter-packs/index.ts` — ขาดไฟล์เดียว = pack ไม่โชว์
และร้านที่เลือกไว้จะ fallback เป็น pack แฟชั่นแทน)

หา royalty-free ได้จาก Unsplash / Pexels (โทนพาสเทล อบอุ่น สว่าง — ให้เข้าชุดกัน)
