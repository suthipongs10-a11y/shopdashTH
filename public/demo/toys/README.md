# รูป starter pack "ของเล่น / แม่และเด็ก"

รูปชุดปัจจุบันเป็น **flat illustration พาสเทลที่ generate จากโค้ด** (`scripts/gen-service-images.mjs`
ส่วนท้ายไฟล์) — ครบตาม `requiredAssets` ของ pack แล้ว ตัวเลือก "ของเล่น / แม่และเด็ก"
จึงเปิดใช้ที่หน้า signup ได้เลย

## อยากเปลี่ยนเป็นรูปถ่ายจริง

วางไฟล์ .webp ชื่อเดิมทับได้ทันที (ระบบอ้างชื่อไฟล์ ไม่สนว่าเป็นภาพวาดหรือภาพถ่าย):

| ไฟล์ | ใช้กับ | สัดส่วน |
|---|---|---|
| `hero-01.webp` | hero หน้าแรก + lookbook | แนวนอน ≥1600px |
| `cat-01..03.webp` | แบนเนอร์หมวด + สไลด์ | 3:2 |
| `toy-01..08.webp` | รูปสินค้าตัวอย่าง 8 ชิ้น | 3:4 |

หา royalty-free ได้จาก Unsplash / Pexels (โทนพาสเทลอบอุ่นให้เข้าชุดธีมเด็ก)
**อย่าลืมจด credit ลง `CREDITS.json`** (รูปแบบเดียวกับ `public/demo/t2/CREDITS.json`)
